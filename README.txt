Vehicle Management System
========================

Installation Instructions:
1. Extract all files to a folder of your choice
2. Double-click VehicleManagement.exe to start the application
3. The application will automatically open in your default web browser
4. If the browser doesn't open automatically, visit http://localhost:5000

Important Notes:
- The application needs to be running to access the system
- Keep the application window open while using the system
- Your data is stored locally in the 'data' folder
- To stop the application, simply close the application window

System Requirements:
- Windows 10 or later
- No additional software required

For Support:
Contact your system administrator or technical support.

Enjoy using the Vehicle Management System!
